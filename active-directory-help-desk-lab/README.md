# Active Directory Help Desk Lab

## Status
In progress.

## Objective
Build a local Windows Server Active Directory environment and practice common help desk and user-access administration tasks.

## Planned Environment
- Windows Server Evaluation
- Windows 11 Evaluation Client
- Active Directory Domain Services
- DNS
- VirtualBox or VMware Workstation

## Planned Tasks
- Create users, groups, and organizational units
- Configure group-based permissions
- Join a Windows client to the domain
- Perform password resets and account unlocks
- Practice basic DNS troubleshooting
- Document troubleshooting steps and screenshots

## Skills Being Developed
Active Directory, Windows administration, DNS, access control, user provisioning, troubleshooting, and technical documentation.
